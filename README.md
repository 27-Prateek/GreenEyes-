# AQI-Sense India — Research Project

Deep learning-based multi-city, multi-step air quality prediction for India.
Extends the GreenEyes WaveNet paper (arXiv:2212.04175) with spatial graph modeling,
multi-step forecasting, automated preprocessing, and rigorous evaluation.

---

## Running the Pipeline

### Option A — Full automatic pipeline

```bash
# Standard: preprocess + train GreenEyes++ + evaluate
python run_all.py

# With hyperparameter search before training (recommended, takes longer)
python run_all.py --search --n_trials 50

# Full ablation: train + evaluate all trained models (excludes wavenet)
python run_all.py --ablation

# Skip preprocessing if already done
python run_all.py --skip_preprocess
```

### Option B — Step by step

```bash
# Step 1: Preprocess (generates sequences via sequence.py internally)
python preprocessing/preprocess.py

# Step 2 (optional): Find best hyperparameters
python training/hyperparameter_search.py --n_trials 50

# Step 3: Train GreenEyes++
python training/train.py --model greeneyes

# Step 3b (optional): Train other baselines
python training/train.py --model lstm
python training/train.py --model gru
python training/train.py --model transformer
# Note: wavenet baseline was not trained in this project

# Step 4: Evaluate (single model)
python evaluation/evaluate.py --model greeneyes

# Step 4b: Ablation study across all trained models
python evaluation/evaluate.py --ablation
```

## Model Architecture: GreenEyes++

```
Input: (batch, 168 hours, ~60 features)
     │
┌────┴────────────────────────────────┐
│  Input Projection                   │  Linear → LayerNorm → GELU → Dropout
└────┬────────────────────────────────┘
     │
┌────┴────────────────────────────────┐
│  WaveNet Backbone (3 blocks)        │  Block 1: 8 dilated layers (dilation 1,2,...,128)
│                                     │  Block 2: 5 dilated layers
│                                     │  Block 3: 3 dilated layers
│                                     │  All with tanh×sigmoid gating + skip connections
└────┬────────────────────────────────┘
     │
┌────┴────────────────────────────────┐
│  Graph Attention Network            │  Multi-head attention across city graph
│                                     │  Edges: cities within 500km, inv-dist weighted
└────┬────────────────────────────────┘
     │
┌────┴────────────────────────────────┐
│  Bidirectional LSTM (2 layers)      │  Captures long-range temporal dependencies
└────┬────────────────────────────────┘
     │
┌────┴────────────────────────────────┐
│  Temporal Attention (custom)        │  Attention(V) = exp(tanh(WV+b)) ⊙ V
│                                     │  Learnable temperature τ (improved over paper)
└────┬────────────────────────────────┘
     │
┌────┴────────────────────────────────┐
│  Shared MLP                         │  2-layer projection with LayerNorm + GELU
└────┬────────────────────────────────┘
     │
┌────┴──────────────┬─────────────────┐
│  Regression Heads │ Classification  │
│  t+1h, t+24h,     │ 6 AQI categories│
│  t+72h  (MSE)     │ (CrossEntropy)  │
└───────────────────┴─────────────────┘
```

**Joint loss:** `L = 0.75 × MSE + 0.25 × CrossEntropy`

## Key Preprocessing Improvements Over Paper

| Step | GreenEyes Paper | This Project |
|---|---|---|
| Dataset | 1 room, 4 sensors, 61h | 5 CSV files, 26 cities, 2015–2024 |
| Polygonalization | Manual hand-labeling | Automated Savitzky-Golay (window=49h) |
| Missing values | Not discussed | 3-tier: interpolate → seasonal median → mask |
| AQI computation | Already available | Recomputed from PM2.5 (EPA formula) |
| Temporal features | Raw timestamps | sin/cos cyclic encoding |
| Lag features | None | Lags at 1, 2, 3, 6, 12, 24, 48, 168h |
| Spatial modeling | None | City graph (inverse-distance edges, 500km threshold) |
| Train/val/test split | Not time-aware | Strict temporal split: train <2019 / val 2019 / test 2020 |
| Sequence building | N/A | Memory-mapped .npy (500 MB chunks, no RAM spike) |

## GPU & Memory Configuration

Tuned for an **RTX 3050 Laptop GPU (4 GB VRAM)**:

| Setting | Value | Notes |
|---|---|---|
| `HIDDEN_DIM` | 64 | Reduced from 128 to fit 4 GB VRAM |
| `BATCH_SIZE` | 32 | Safe for HIDDEN_DIM=64 |
| `GRAD_ACCUM_STEPS` | 4 | Effective batch = 128 |
| `USE_AMP` | True | Mixed precision — halves VRAM usage |

To use `HIDDEN_DIM=128`, enable `USE_AMP=True` and/or reduce `BATCH_SIZE`.

## Configuration

All hyperparameters are in `configs/config.py`. Key ones:

```python
WINDOW_SIZE      = 168       # 7 days of hourly input
FORECAST_HOURS   = [1, 24, 72]
HIDDEN_DIM       = 64        # 128 if using AMP on larger GPU
WAVENET_LAYERS   = [8, 5, 3]
EPOCHS           = 4         # increase to 100–150 for full training
BATCH_SIZE       = 32
GRAD_ACCUM_STEPS = 4         # effective batch = 128
LEARNING_RATE    = 1e-3
PATIENCE         = 15
LOSS_ALPHA       = 0.75      # regression weight
LOSS_BETA        = 0.25      # classification weight
USE_AMP          = True      # automatic mixed precision
```
